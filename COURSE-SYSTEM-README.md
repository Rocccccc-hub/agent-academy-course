## 🐛 故障排除

### 视频无法播放

**问题**: 视频播放器显示黑屏

**解决**:
1. 检查视频文件是否在 `/Users/anker/Desktop/CC project/lessons/` 目录
2. 确认文件名为 `DAY1-agent 架构.mp4`
3. 刷新浏览器页面
4. 尝试其他浏览器（Chrome/Safari/Firefox）

### 进度不保存

**问题**: 学习进度丢失

**解决**:
1. 检查浏览器是否启用 localStorage
2. 不要使用无痕/隐私模式
3. 清除浏览器缓存后重试

### 章节内容不显示

**问题**: 点击章节无反应

**解决**:
1. 确认 `course-data.js` 文件存在
2. 打开浏览器控制台查看错误
3. 刷新页面重试
